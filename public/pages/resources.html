Learning Resources

$ cat resources.md

Curated list of the best tools, platforms, books, and websites for learning cybersecurity and networking.

$ Practicing Platforms

Platform · Free / Paid

TryHackMe

Guided learning paths for beginners to advanced. Covers web hacking, networking, OSINT, and more. Best place to start.
Platform · Free / Paid

Hack The Box

Real-world penetration testing challenges and labs. More advanced than TryHackMe. Great for CTF prep.
Platform · Free

PortSwigger Web Academy

The definitive resource for web application security. Labs covering every OWASP vulnerability. Made by the creators of Burp Suite.
Platform · Free

OverTheWire

Classic wargames for learning Linux, networking, and security fundamentals through challenges. Start with Bandit.
CTF · Free

picoCTF

Beginner-friendly CTF competition by Carnegie Mellon University. Great for learning the fundamentals of security challenges.
Platform · Free

VulnHub

Download vulnerable virtual machines to practice on locally. No internet needed — perfect for offline lab practice.

$ Essential Tools

Tool · Free / Open Source

Nmap

The essential network scanner. Host discovery, port scanning, service detection, OS fingerprinting, and scripting.
Tool · Free / Open Source

Wireshark

The world's foremost network protocol analyzer. Capture and analyze network traffic in real time.
Tool · Free / Paid

Burp Suite

The standard tool for web application security testing. Intercept, modify, and replay HTTP requests.
Tool · Free / Open Source

Metasploit Framework

The world's most used penetration testing framework. Thousands of exploits, payloads, and post-exploitation modules.
Tool · Free / Open Source

GNS3

Network simulation software for building and testing virtual networks. Run real Cisco IOS images in your lab.
Tool · Free / Open Source

pfSense

Open-source firewall and router. Perfect for building a home lab firewall with real enterprise features.

$ Learning Resources

Website · Free

OWASP

The Open Web Application Security Project. Source of the OWASP Top 10, testing guides, and security checklists.
Courses · Free / Paid

Cybrary

Free cybersecurity courses covering ethical hacking, network security, incident response, and certifications prep.
Certification · Paid

Cisco CCNA

The gold standard entry-level networking certification. Covers routing, switching, security, and automation fundamentals.
YouTube · Free

NetworkChuck

Engaging videos on networking, hacking, Linux, and certifications. Great entry point for beginners.
Reading Room · Free

SANS Reading Room

Thousands of free whitepapers and research papers on every aspect of information security and networking.
Database · Free

Exploit-DB

Maintained by Offensive Security. Archive of public exploits and vulnerable software for research purposes.
